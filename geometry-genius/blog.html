3: What is event bubble?

When an event is triggered in a DOM, the action of this event works through some steps from where it was triggered until it reaches the user interface. And that is called event bubbling.

What is a callback function and why will you use it?

When a function is passed as an argument to another function, it is called a callback function. We use callback function to execute something in one function in another function.
